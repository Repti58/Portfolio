Изучаю frontend разработку с конца 2021 года. Ранее имел опыт коммерческой разработки корпоративных сайтов, сайтов визиток и лендингов. Параллельно с этим занимался разработкой фирменных логотипов и полиграфической продукции (визитки, открытки, рекламные буклеты). На тот момент освоил работу в Adobe Photoshop, CorelDRAW. Так как база в виде HTML и CSS уже была, изучение Frontend начал сразу с JavaScript. Затем приступил к изучению React и Redux.

За время обучения приобрёл следующие навыки:
  • Работа с Git;
  • Опыт разработки Backend на Node.js + Express.
  • Опыт написания запросов к БД (SQL-lite, MongoDB).
  • Deploy backend и frontend на серверах, таких сервисов как: GitHub, Vercell, Glitch
  • Дебаггинг чистого JavaScript и React/Redux приложений.
  • Адаптивная верстка
